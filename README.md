# Smart Acne Severity Grading System for Diverse Skin Types

## 📌 Overview

The **Smart Acne Severity Grading System** is an AI-powered solution designed to automatically detect, classify, and assess acne severity from facial images. The system leverages advanced deep learning architectures to provide objective and consistent evaluation, reducing reliance on manual clinical assessment.

---

## 🚀 Features

* 🔍 **Automated Acne Detection** from facial images
* 📊 **Severity Grading System** for different acne levels
* 🧠 **Hybrid Deep Learning Model** using SkinSwin-T and CapsNet
* 🎯 **High Accuracy Classification** across diverse skin types
* 🤖 **AI Assistant** for personalized skincare recommendations

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Deep Learning Frameworks:** TensorFlow, Keras
* **Computer Vision:** OpenCV
* **Models Used:**

  * SkinSwin Transformer (SkinSwin-T)
  * Capsule Networks (CapsNet)

---

## 🧩 System Architecture

1. **Image Input** – User uploads facial image
2. **Preprocessing** – Image normalization and enhancement using OpenCV
3. **Feature Extraction** – SkinSwin-T extracts high-level features
4. **Classification** – CapsNet performs acne type classification
5. **Severity Scoring** – Model outputs acne severity level
6. **Recommendation Engine** – AI assistant suggests skincare guidance
